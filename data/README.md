## Data Simulasi Kredit SME

Folder ini berisi data simulasi (dummy data) pengajuan kredit SME yang digunakan
dalam proyek analisis pipeline kredit.

Data dibuat untuk tujuan pembelajaran dan portofolio, serta disesuaikan dengan
alur proses administrasi kredit SME secara high-level berdasarkan observasi terbatas
selama masa magang di Unit Kredit SME.

Seluruh data tidak merepresentasikan data riil institusi perbankan manapun.
